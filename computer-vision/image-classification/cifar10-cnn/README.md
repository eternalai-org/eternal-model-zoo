# nft-mlp

A simple CNN model trained using the popular [CIFAR-10](https://paperswithcode.com/dataset/cifar-10) dataset, that can classify between photorealistic images of the following classes:
- airplane
- automobile
- bird
- cat
- deer
- dog
- frog
- horse
- ship
- truck
